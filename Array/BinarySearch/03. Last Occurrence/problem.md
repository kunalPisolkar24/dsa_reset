# 3. Last Occurrence

**Difficulty:** Medium

## Problem Statement

Given an array of integers `nums` sorted in non-decreasing order, find the index of the **last occurrence** of a given `target` value. If the `target` is not found in the array, return `-1`.

### Example Testcases

**Example 1:**
```
Input: nums = [1, 2, 2, 2, 3], target = 2
Output: 3
```

**Example 2:**
```
Input: nums = [1, 2, 2, 2, 3], target = 4
Output: -1
```

## Constraints

- `1 <= nums.length <= 10^4`
- `-10^4 < nums[i], target < 10^4`
- `nums` is sorted in non-decreasing order.

Problem: Dice Combinations

You are given an integer N.

You have a standard 6-faced dice with values:
1, 2, 3, 4, 5, 6

Count the number of distinct ordered ways to reach sum N.

Since the answer can be very large, return it modulo 1e9 + 7.

--------------------------------------------------

Input:
A single integer N

Output:
Print the number of ordered ways to form sum N.

--------------------------------------------------

Constraints:

1 <= N <= 1e6

--------------------------------------------------

Example 1

Input:
3

Output:
4

Explanation:
1+1+1
1+2
2+1
3

--------------------------------------------------

Example 2

Input:
4

Output:
8

--------------------------------------------------

Example 3

Input:
7

Output:
63

--------------------------------------------------

Example 4

Input:
10

Output:
492

--------------------------------------------------

Example 5

Input:
1

Output:
1

# 6. Largest Component

**Difficulty:** Medium

## Problem Statement
Return the size of the largest connected component in the given graph.

### Example Testcases

**Example 1:**
```
Input:
n = 18
edges = [[10, 15], [5, 15], [1, 7], [10, 12]]
Output: 4
```

**Example 2:**
```
Input:
n = 3
edges = [[0, 2]]
Output: 2
```

## Constraints
- `1 <= n <= 10^5`
- `0 <= edges.length <= 2 * 10^5`
- `edges[i].length == 2`

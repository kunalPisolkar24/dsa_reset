# 4. Count Connected Components

**Difficulty:** Medium

## Problem Statement
Given `n` nodes and `edges`, count how many distinct connected components exist.

### Example Testcases

**Example 1:**
```
Input:
n = 2
edges = []
Output: 2
```

**Example 2:**
```
Input:
n = 5
edges = []
Output: 5
```

## Constraints
- `1 <= n <= 10^5`
- `0 <= edges.length <= 2 * 10^5`
- `edges[i].length == 2`

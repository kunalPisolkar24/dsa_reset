# 2. Maximum Valid Square

**Difficulty:** Very Easy

## Problem Statement

Given a non-negative integer `x`, find the **largest integer y** such that `y * y <= x`.

### Example Testcases

**Example 1:**
```
Input: x = 27
Output: 5
Explanation: 5*5 = 25 (<= 27). 6*6 = 36 (not <= 27). Thus, 5 is the largest integer.
```

**Example 2:**
```
Input: x = 25
Output: 5
Explanation: 5*5 = 25 (<= 25).
```

## Constraints

- `0 <= x <= 2^31 - 1`

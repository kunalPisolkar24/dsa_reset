# 9. First Taller Building

**Difficulty:** Medium

## Problem Statement

A row of buildings has different heights. For each building, find the height of the nearest building to its right that is strictly taller.

### Example Testcases

**Example 1:**
```
Input: heights = {3, 7, 8, 3, 6, 1}
Output: {7, 8, -1, 6, -1, -1}
```

**Example 2:**
```
Input: heights = {1, 2, 3, 4}
Output: {2, 3, 4, -1}
```

**Example 3:**
```
Input: heights = {5, 4, 3, 2, 1}
Output: {-1, -1, -1, -1, -1}
```

## Constraints

- `1 <= n <= 10^5` 
- `1 <= heights[i] <= 10^9`

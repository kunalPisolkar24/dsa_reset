# 14. Sum of Subarray Maximums

**Difficulty:** Hard

## Problem Statement

Given an array of integers, find the sum of `max(b)`, where `b` ranges over every (contiguous) subarray of the array.

### Example Testcases

**Example 1:**
```
Input: arr = {3, 1, 2, 4}
Output: 30
```

**Example 2:**
```
Input: arr = {1, 2, 3}
Output: 14
```

**Example 3:**
```
Input: arr = {1, 1, 1}
Output: 6
```

## Constraints

- `1 <= n <= 3 * 10^4` 
- `1 <= arr[i] <= 3 * 10^4`

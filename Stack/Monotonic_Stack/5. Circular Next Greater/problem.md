# 5. Circular Next Greater

**Difficulty:** Easy

## Problem Statement

The array is treated as circular, meaning the search continues from the end back to the beginning. Find the first greater element for each index.

### Example Testcases

**Example 1:**
```
Input: arr = {1, 2, 1}
Output: {2, -1, 2}
```

**Example 2:**
```
Input: arr = {5, 4, 3, 2, 1}
Output: {-1, 5, 5, 5, 5}
```

**Example 3:**
```
Input: arr = {1, 5, 3, 6, 8}
Output: {5, 6, 6, 8, -1}
```

## Constraints

- `1 <= n <= 10^5` 
- `-10^9 <= arr[i] <= 10^9`

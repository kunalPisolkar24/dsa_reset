# 1. Reverse Using Stack

**Difficulty:** Easy

## Problem Statement

You are given an array of integers. Reverse the array using a **stack** (no in-place swapping allowed).

### Example Testcases

**Example 1:**
```
Input: arr = [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
```

**Example 2:**
```
Input: arr = [10]
Output: [10]
```

## Constraints

- `0 <= n <= 10^5`
- `-10^9 <= arr[i] <= 10^9`
- You must use a stack explicitly.

# 2. Check Bit

**Difficulty:** Easy

## Problem Statement

Given a non-negative integer `n` and a non-negative integer `i`, return `true` if the `i`-th bit of `n` is set. Otherwise, return `false`.

The least significant bit is bit `0`.

### Example Testcases

**Example 1:**
```
Input: n = 13, i = 2
Output: true
Explanation: 13 in binary is 1101, and bit 2 is 1.
```

**Example 2:**
```
Input: n = 13, i = 1
Output: false
Explanation: 13 in binary is 1101, and bit 1 is 0.
```

## Constraints

- `0 <= n <= 10^9`
- `0 <= i <= 30`

# 1. Odd or Even

**Difficulty:** Easy

## Problem Statement

Given an integer `n`, return `true` if `n` is odd. Otherwise, return `false`.

### Example Testcases

**Example 1:**
```
Input: n = 7
Output: true
Explanation: 7 is not divisible by 2, so it is odd.
```

**Example 2:**
```
Input: n = 12
Output: false
Explanation: 12 is divisible by 2, so it is even.
```

## Constraints

- `-10^9 <= n <= 10^9`

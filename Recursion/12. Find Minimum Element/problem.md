# 12. Find Minimum Element

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, return the minimum element using **recursion**. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: arr = [5, 1, 7, 2, 9]
Output: 1
Explanation: 1 is the minimum element in the array.
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i] <= 10^9`

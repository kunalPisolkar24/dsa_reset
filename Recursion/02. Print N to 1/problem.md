# 2. Print N to 1

**Difficulty:** Easy

## Problem Statement

Given an integer `N`, write a function to return an array containing all numbers from `N` down to `1` in descending order. You must use **recursion** to solve this problem. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: N = 5
Output: [5, 4, 3, 2, 1]
Explanation: The numbers from 5 to 1 are printed in descending order.
```

**Example 2:**
```
Input: N = 1
Output: [1]
Explanation: Only 1 is printed.
```

## Constraints

- `1 <= N <= 10^4`

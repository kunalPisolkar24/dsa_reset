# 15. Check if Array is Palindrome

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, check whether the array reads the same forward and backward using **recursion**. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: arr = [1, 2, 3, 2, 1]
Output: true
```

**Example 2:**
```
Input: arr = [1, 2, 3, 4]
Output: false
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i] <= 10^9`

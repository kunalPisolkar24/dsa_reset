# 3. Print Even Numbers from 1 to N

**Difficulty:** Easy

## Problem Statement

Given an integer `N`, return an array containing all even numbers from `1` to `N` (inclusive) in ascending order. You must use **recursion** to solve this problem. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: N = 10
Output: [2, 4, 6, 8, 10]
Explanation: The even numbers from 1 to 10 are printed in ascending order.
```

**Example 2:**
```
Input: N = 3
Output: [2]
Explanation: The only even number between 1 and 3 is 2.
```

## Constraints

- `1 <= N <= 10^4`

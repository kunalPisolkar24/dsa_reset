# 22. Convert Integer to Binary

**Difficulty:** Easy

## Problem Statement

Given a non-negative integer `N`, return its binary representation as a string using **recursion**. Do not use any loops or built-in conversion functions like `std::bitset`.

### Example Testcases

**Example 1:**
```
Input: 10
Output: "1010"
```

**Example 2:**
```
Input: 0
Output: "0"
```

## Constraints

- `0 <= N <= 10^9`

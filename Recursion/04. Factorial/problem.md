# 4. Factorial

**Difficulty:** Easy

## Problem Statement

Given a positive integer `N`, compute its factorial `N!` using **recursion**. Do not use any loops.

The factorial of a number `N` is defined as:
`N! = N * (N - 1) * ... * 1`

For `N = 0`, the factorial is `1`.

### Example Testcases

**Example 1:**
```
Input: N = 5
Output: 120
Explanation: 5! = 5 * 4 * 3 * 2 * 1 = 120
```

**Example 2:**
```
Input: N = 0
Output: 1
Explanation: 0! is defined as 1.
```

## Constraints

- `0 <= N <= 20`

# 19. Remove All Occurrences of X from Array

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers and a value `x`, remove all occurrences of `x` using **recursion** and return a new array containing the remaining elements in their original order. Do not use any loops (`for`, `while`, etc.).

### Example Testcases

**Example 1:**
```
Input: arr = [1, 3, 5, 3, 1], x = 3
Output: [1, 5, 1]
```

**Example 2:**
```
Input: arr = [2, 2, 2], x = 2
Output: []
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i], x <= 10^9`

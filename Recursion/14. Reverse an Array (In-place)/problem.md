# 14. Reverse an Array (In-place)

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, reverse it **in-place** using **recursion**. Do not use any loops or the built-in `reverse()` function.

*Hint: Use two pointers (`left`, `right`) and pass them recursively.*

### Example Testcases

**Example 1:**
```
Input: arr = [1, 2, 3, 4]
Output: [4, 3, 2, 1]
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i] <= 10^9`

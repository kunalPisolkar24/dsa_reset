# 8. Count Zeros in a Number

**Difficulty:** Easy

## Problem Statement

Given an integer `N`, count how many `0`s are present in it using **recursion**. Do not use any loops or string conversions.

### Example Testcases

**Example 1:**
```
Input: N = 102030
Output: 3
Explanation: The digits of N contain three 0s.
```

**Example 2:**
```
Input: N = 5
Output: 0
Explanation: No zeros.
```

## Constraints

- `-10^9 <= N <= 10^9`

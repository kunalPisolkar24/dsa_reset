# 1. Print 1 to N

**Difficulty:** Easy

## Problem Statement

Given an integer `N`, write a function to return an array containing all numbers from `1` to `N` in ascending order. You must use **recursion** to solve this problem. Do not use any loops (`for`, `while`, etc.).

### Example Testcases

**Example 1:**
```
Input: N = 5
Output: [1, 2, 3, 4, 5]
Explanation: The numbers from 1 to 5 are printed in ascending order.
```

**Example 2:**
```
Input: N = 1
Output: [1]
Explanation: Only 1 is printed.
```

## Constraints

- `1 <= N <= 10^4`

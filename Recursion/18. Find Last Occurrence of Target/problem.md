# 18. Find Last Occurrence of Target

**Difficulty:** Easy

## Problem Statement

Return the **last index** of the `target` in the given array using **recursion**. Return `-1` if the target is not found. Do not use loops.

### Example Testcases

**Example 1:**
```
Input: arr = [1, 3, 3, 7], target = 3
Output: 2
```

**Example 2:**
```
Input: arr = [1, 2, 4], target = 5
Output: -1
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i], target <= 10^9`

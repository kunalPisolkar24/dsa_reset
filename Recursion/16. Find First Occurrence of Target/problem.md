# 16. Find First Occurrence of Target

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers and a `target`, return the **index of the first occurrence** of the target using **recursion**.
Return `-1` if not found. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: arr = [1, 5, 3, 7], target = 3
Output: 2
```

**Example 2:**
```
Input: arr = [1, 2, 3], target = 5
Output: -1
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i], target <= 10^9`

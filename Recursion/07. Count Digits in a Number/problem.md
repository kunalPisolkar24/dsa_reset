# 7. Count Digits in a Number

**Difficulty:** Easy

## Problem Statement

Given an integer `N`, count the total number of digits it has using **recursion**. Do not use any loops or string conversions.

### Example Testcases

**Example 1:**
```
Input: N = 12345
Output: 5
Explanation: The number has 5 digits.
```

**Example 2:**
```
Input: N = 0
Output: 1
Explanation: 0 has 1 digit.
```

## Constraints

- `-10^9 <= N <= 10^9`

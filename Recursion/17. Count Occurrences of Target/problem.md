# 17. Count Occurrences of Target

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers and a `target`, count how many times `target` appears in the array using **recursion**. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: arr = [1, 3, 3, 5, 3], target = 3
Output: 3
```

**Example 2:**
```
Input: arr = [1, 2, 4], target = 5
Output: 0
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i], target <= 10^9`

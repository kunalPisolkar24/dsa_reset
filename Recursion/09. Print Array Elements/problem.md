# 9. Print Array Elements

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, write a function to return an array containing all its elements in order using **recursion**. Do not use any loops (`for`, `while`, etc.).

### Example Testcases

**Example 1:**
```
Input: arr = [1, 2, 3, 4, 5]
Output: [1, 2, 3, 4, 5]
Explanation: Elements are collected in the same order.
```

## Constraints

- `1 <= N <= 10^4`
- `-10^4 <= arr[i] <= 10^4`

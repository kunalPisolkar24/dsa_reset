# 6. Sum of Digits of a Number

**Difficulty:** Easy

## Problem Statement

Given an integer `N`, return the sum of its digits using **recursion**. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: N = 1234
Output: 10
Explanation: 1 + 2 + 3 + 4 = 10
```

**Example 2:**
```
Input: N = 987
Output: 24
Explanation: 9 + 8 + 7 = 24
```

## Constraints

- `-10^9 <= N <= 10^9`

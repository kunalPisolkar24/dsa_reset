# 21. Reverse a String

**Difficulty:** Easy

## Problem Statement

Given a string `s`, return a **new** string that is its reverse using **recursion**. Do not modify the original string or use built-in matching reverse functions. Do not use loops.

### Example Testcases

**Example 1:**
```
Input: s = "hello"
Output: "olleh"
```

**Example 2:**
```
Input: s = "recursion"
Output: "noisrucer"
```

## Constraints

- `1 <= s.length <= 10^4`
- `s` consists of printable ASCII characters.

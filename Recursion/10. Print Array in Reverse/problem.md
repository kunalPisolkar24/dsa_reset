# 10. Print Array in Reverse

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, return an array containing all its elements in **reverse order** using recursion. Do not use loops.

### Example Testcases

**Example 1:**
```
Input: arr = [1, 2, 3, 4, 5]
Output: [5, 4, 3, 2, 1]
```

## Constraints

- `1 <= N <= 10^4`
- `-10^4 <= arr[i] <= 10^4`

# 5. Sum of First N Natural Numbers

**Difficulty:** Easy

## Problem Statement

Given a positive integer `N`, compute the sum of the first `N` natural numbers using **recursion**. Do not use any loops or math formulas.

### Example Testcases

**Example 1:**
```
Input: N = 5
Output: 15
Explanation: 1 + 2 + 3 + 4 + 5 = 15
```

**Example 2:**
```
Input: N = 1
Output: 1
Explanation: The sum of the first 1 natural number is 1.
```

## Constraints

- `1 <= N <= 10^4`

# 11. Find Maximum Element

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, return the maximum element using **recursion**. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: arr = [5, 1, 7, 2, 9]
Output: 9
Explanation: 9 is the maximum element in the array.
```

**Example 2:**
```
Input: arr = [5]
Output: 5
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i] <= 10^9`

# 13. Check if Array is Sorted

**Difficulty:** Easy

## Problem Statement

Given an array of `N` integers, check whether it is sorted in **non-decreasing order** using **recursion**. Do not use any loops.

### Example Testcases

**Example 1:**
```
Input: arr = [1, 2, 3, 4, 5]
Output: true
```

**Example 2:**
```
Input: arr = [1, 3, 2, 5]
Output: false
```

## Constraints

- `1 <= N <= 10^4`
- `-10^9 <= arr[i] <= 10^9`

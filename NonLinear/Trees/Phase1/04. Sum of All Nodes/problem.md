# 4. Sum of All Nodes

**Difficulty:** Easy

## Problem Statement

Given the `root` of a binary tree, return the **sum of all node values** in the tree.

## Example Testcases

**Example 1:**
```
Input: root = [1,2,3,4]
Output: 10
Explanation: 1 + 2 + 3 + 4 = 10.
```

**Example 2:**
```
Input: root = [10,-2,5]
Output: 13
Explanation: 10 + (-2) + 5 = 13.
```

**Example 3:**
```
Input: root = []
Output: 0
```

## Constraints

- The number of nodes in the tree is in the range `[0, 10^4]`.
- `-1000 <= Node.val <= 1000`

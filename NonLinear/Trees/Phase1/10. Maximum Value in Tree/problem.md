# 10. Maximum Value in Tree

**Difficulty:** Easy

## Problem Statement

Given the `root` of a binary tree, return the **maximum value** among all nodes.

## Example Testcases

**Example 1:**
```
Input: root = [1,3,2,5]
Output: 5
```

**Example 2:**
```
Input: root = [-10,-3,-20]
Output: -3
```

**Example 3:**
```
Input: root = [1]
Output: 1
```

## Constraints

- The number of nodes in the tree is in the range `[1, 10^4]`.
- `-10^4 <= Node.val <= 10^4`
